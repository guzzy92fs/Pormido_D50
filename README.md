## üõ† Installation Instructions

Follow these steps carefully to ensure a successful update. **Do not power off the device during the update process.**

### 1. Prepare the SD Card
* Use an SD card with a capacity between **8 GB and 32 GB**.
* **Important:** Backup any existing footage or data on the card before proceeding.
* **Format** the SD card (FAT32 is generally recommended for dashcams).

### 2. Copy the Firmware
* Connect your SD card to your computer or iPad using a card reader.
* Copy the `SD_CarDV.bin` file from this repository directly to the **root directory** of your SD card.
* Ensure no other files are on the card except for `SD_CarDV.bin`.

### 3. Update the Dashcam
* Insert the SD card into the Pormido D50 mirror.
* Power on the mirror (or ensure it is connected to power).
* The system will detect the file and update **automatically**.
* The process typically takes about **2 minutes**.

### 4. Verify the Update
* Once the device reboots, go to the system settings.
* Check the firmware version. If successful, it should display: **20240101_V0.1**.
